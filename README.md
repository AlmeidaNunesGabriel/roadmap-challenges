#  Roadmap.sh Challenges

Este repositório é um hub central para todos os projetos e desafios práticos que estou a realizar com base nos guias do [roadmap.sh](https://roadmap.sh/). O objetivo é consolidar conhecimentos fundamentais e explorar novas tecnologias de forma estruturada.

## Organização do Repositório

Os projetos estão organizados por domínio por área(inicialmente só Frontend) para facilitar a navegação. Cada subpasta contém o código-fonte, documentação específica e instruções de execução do respetivo desafio.

## 🎯 Objetivos Técnicos

- **Reciclagem Técnica:** Reforçar os fundamentos de desenvolvimento de software.
- **Domínio de Ecossistemas:** Praticar com JavaScript moderno, Node.js e ferramentas de versionamento.
- **Portefólio Unificado:** Manter um histórico evolutivo de soluções em um único local, demonstrando consistência.

---

## ⚙️ Como executar os projetos

Como este é um **monorepo**, cada projeto pode ter as suas próprias dependências. Para correr qualquer um deles:

1. Navegue até a pasta do projeto: `cd categoria/nome-do-projeto`
2. Instale as dependências: `npm install` (se aplicável)
3. Execute conforme as instruções no README interno da pasta.
